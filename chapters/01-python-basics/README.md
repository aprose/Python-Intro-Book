# Chapter 1: Python Basics

This chapter introduces you to the fundamentals of Python programming.

## Topics Covered

1. **Introduction to Python**
   - What is Python?
   - Why Python for programming and data science?
   - Setting up your Python environment

2. **Variables and Data Types**
   - Integers, Floats, Strings, Booleans
   - Type conversion
   - Variable naming conventions

3. **Control Flow**
   - if-elif-else statements
   - for loops
   - while loops
   - break and continue

4. **Functions**
   - Defining functions
   - Parameters and return values
   - Scope and namespaces

5. **Basic Input/Output**
   - print() function
   - input() function
   - String formatting

## Files in This Chapter

- `01_introduction.ipynb` - Introduction and setup
- `02_variables_datatypes.ipynb` - Variables and data types
- `03_control_flow.ipynb` - Conditional statements and loops
- `04_functions.ipynb` - Functions and scope
- `05_input_output.ipynb` - Input/output operations

## Learning Objectives

By the end of this chapter, you will be able to:
- Write basic Python programs
- Use variables and understand different data types
- Implement control flow logic
- Create and use functions
- Handle basic input and output

## Prerequisites

- No prior programming experience required
- Python 3.8+ installed on your system
