# Chapter 5: Deep Learning Basics

This chapter introduces the fundamentals of deep learning and neural networks.

## Topics Covered

1. **Introduction to Machine Learning**
   - Supervised vs Unsupervised Learning
   - Training, Validation, and Test Sets
   - Overfitting and Underfitting
   - Model Evaluation Metrics

2. **Neural Networks Fundamentals**
   - Biological inspiration
   - Mathematical representation
   - Forward propagation
   - Backpropagation algorithm

3. **Perceptrons and Activation Functions**
   - Single-layer perceptron
   - Multi-layer perceptron (MLP)
   - Activation functions (Sigmoid, ReLU, Tanh, Softmax)
   - Loss functions

4. **Introduction to TensorFlow and PyTorch**
   - Framework comparison
   - Building your first neural network in PyTorch
   - Building your first neural network in TensorFlow
   - Training loops and optimization

## Files in This Chapter

- `01_ml_introduction.ipynb` - Machine learning basics
- `02_neural_network_fundamentals.ipynb` - Neural network theory
- `03_activation_functions.ipynb` - Activation functions and perceptrons
- `04_pytorch_intro.ipynb` - Getting started with PyTorch
- `05_tensorflow_intro.ipynb` - Getting started with TensorFlow

## Learning Objectives

By the end of this chapter, you will be able to:
- Understand the fundamentals of machine learning
- Explain how neural networks work
- Implement activation functions
- Build basic neural networks using PyTorch and TensorFlow
- Train models on simple datasets

## Prerequisites

- Completion of Chapter 4: Data Science Introduction
- Understanding of basic linear algebra and calculus
- Familiarity with NumPy and Pandas
