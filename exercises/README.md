# Python Basics Exercises

This directory contains practice exercises for learning Python fundamentals.

## Chapter 1: Python Basics

### Exercise 1: Variables and Arithmetic
Create a Python script that:
1. Defines variables for length and width of a rectangle
2. Calculates the area and perimeter
3. Prints the results with descriptive messages

### Exercise 2: Control Flow
Write a program that:
1. Takes a number as input
2. Checks if it's positive, negative, or zero
3. Checks if it's even or odd
4. Prints appropriate messages

### Exercise 3: Loops
Create a program that:
1. Prints all even numbers from 1 to 20
2. Calculates the sum of numbers from 1 to 100
3. Prints a multiplication table for a given number

### Exercise 4: Functions
Write functions for:
1. Converting Celsius to Fahrenheit
2. Checking if a number is prime
3. Calculating factorial of a number
4. Finding the maximum of three numbers

## Chapter 5: Deep Learning Basics

### Exercise 1: NumPy Basics
Practice with NumPy:
1. Create arrays of different shapes
2. Perform matrix operations
3. Use broadcasting
4. Apply array indexing and slicing

### Exercise 2: Data Preparation
Create a data preprocessing pipeline:
1. Load a dataset (e.g., from sklearn)
2. Split into train/validation/test sets
3. Normalize the features
4. Handle missing values

### Exercise 3: Simple Perceptron
Implement a perceptron from scratch:
1. Create the perceptron class
2. Implement forward pass
3. Implement weight updates
4. Train on a simple linearly separable dataset

## Solutions

Solutions to all exercises can be found in the `solutions/` subdirectory.

## Tips

- Start with simple exercises and gradually increase complexity
- Test your code frequently
- Don't look at solutions immediately - try to solve problems on your own first
- Experiment with variations of each exercise
